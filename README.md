# JobRadar
Система автоматического сбора вакансий с нескольких сайтов, индексирующая их и предоставляющая поисковый интерфейс по ключевым словам, должности и городу.


## Quick Start

```bash
# 1. Установка всех библиотек
poetry install

# 2. Если стоит Poetry 2.0+, сначала установите плагин:
poetry self add poetry-plugin-shell

# 3. Затем активируйте окружение:
poetry shell

# 4. Запуск проекта
python main.py
```
